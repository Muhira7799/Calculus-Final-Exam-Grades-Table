# Calculus Final Exam Grades Table

## Overview
This project demonstrates how to create and organize data using an HTML table. It displays students' final exam grades in Calculus and includes a table header, body, footer, and the average grade.

## Features
- Structured HTML table
- Table caption
- Header, body, and footer sections
- Student names and grades
- Average grade displayed in the footer

## Technologies Used
- HTML5

## What I Learned
While building this project, I learned how to:
- Create tables using the `<table>` element.
- Add a title with the `<caption>` element.
- Organize data using `<thead>`, `<tbody>`, and `<tfoot>`.
- Use `<th>` for table headers.
- Merge table cells using the `colspan` attribute.
